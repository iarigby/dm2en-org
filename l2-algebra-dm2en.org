**** Grupoid
     Let $G$ be a set, if $\otimes:G\times G \to G$ is a binary operation on
     $G$, then the pair $(G,\otimes)$ is a *groupoud*.
**** Semigroup
     Let $(G,\otimes)$ be a groupoid, if the operation is associative
     \ie $a \otimes (b \otimes c) = (a \otimes b) \otimes c$ for every
     element $a,b,c\in G$, then $(G,\otimes)$ is a *semigroup*.
**** Monoid
